# DOM

## DOM结构——两个节点之间可能存在哪些关系以及如何在节点之间任意移动

同级（在前，在后）、父子、包含

移动方法：
| 添加 | appendChild |
| 删除 | removeChild |
| 替换 | replaceChild |
| 插入 | insertBefore |

一个节点不能同时位于文档的两个点中。因此，如果被插入节点已经有父节点，则首先删除该节点，然后将其插入到新位置

若要保留已在文档中的被插入节点，在将该节点追加到新父节点之前，可以使用 Node.cloneNode() 复制节点。
注意，使用 cloneNode() 创建的节点副本不会自动与原始节点保持同步。
